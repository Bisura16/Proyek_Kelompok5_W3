## APLIKASI SCRAPING BERITA DENGAN GUI (SELENIUM)
Aplikasi ini merupakan program web scraping berita berbasis Python dengan antarmuka GUI menggunakan PyQt. Program ini memungkinkan pengguna memasukkan URL halaman berita, kemudian sistem akan mengambil seluruh artikel dari halaman tersebut, membuka setiap artikel, dan mengekstrak informasi berupa judul berita, tanggal publikasi, serta isi berita. Hasil scraping kemudian ditampilkan dalam tabel pada aplikasi.
